first step

It is very important to configure Git to use your Name and email. For Git to know who you are and to track your changes it is necessary to set the following settings an all hosts!: It is important to use the same values as the sysadmins use for managing your account!

to get the full name from ldap you can run the following command:

ldapsearch -x -LLL "uid=$USER" displayname  2>/dev/null | perl -pe 's/displayName:: (.*)/echo \1 | base64 -d/; s/displayName: /echo /; s/dn.*//; $_="" if m/^$/;' | bash

and to get the configured email you can run the command:

ldapsearch -x -LLL "uid=$USER" mail  2>/dev/null | awk '/mail/ {print $2}'

to configure git run the following commands:

git config --global user.name "Your Full Name"
git config --global user.email "your email"

to get the values directly from ldap for example:

git config --global user.name "$(ldapsearch -x -LLL "uid=$USER" displayname  2>/dev/null | perl -pe 's/displayName:: (.*)/echo \1 | base64 -d/; s/displayName: /echo /; s/dn.*//; $_="" if m/^$/' | bash)"
git config --global user.email "$(ldapsearch -x -LLL uid=$USER mail  2>/dev/null | awk '/mail/ {print $2}')"

using git via ssh (at work or at home)

If you generate a SSH key you are able to use git at work or at home without always have to enter a password. The following procedures have been successfully tested in Linux and Mac OS X.

generate SSH key

run the following command (replace the email with your email)

ssh-keygen -t rsa -C "your_email@example.com"

choose a filename e.g.:

/afs/spsc.tugraz.at/user/username/.ssh/gitint.spsc.tugraz.at-id_rsa

IMPORTANT: do not enter a passphrase otherwise you always have to enter it!

configure ssh to use the key for the server gitint.spsc.tugraz.at

edit the file ~/.ssh/config (if it does not exist create it by using the command touch ~/.ssh/config) and add the following content (enter the right path for the rsa key), e.g., by using mcedit, pico, vim, etc.:

Host gitint gitint.spsc.tugraz.at
    HostName gitint.spsc.tugraz.at
    User git
    ForwardX11 no
    IdentityFile ~/.ssh/gitint.spsc.tugraz.at-id_rsa

Host gitintdev gitintdev.spsc.tugraz.at
    HostName gitintdev.spsc.tugraz.at
    User git
    ForwardX11 no
    IdentityFile ~/.ssh/gitint.spsc.tugraz.at-id_rsa

IMPORTANT: change the permission of the config file with chmod 700 ~/.ssh/config

Add your SSH key to GitLab

get your public SSH key (enter the right path for the rsa key):

cat ~/.ssh/gitint.spsc.tugraz.at-id_rsa.pub

go to profile/keys press Add SSH Key and copy the output in the form fild Key at the GitLab page. Enter the name of the rsa file as Title and Press Add key.
